Game of Life

Developed by Ben Duncan
View source

Conway's Game of Life is a simple computer simulation of life in which black squares represent living cells. The simulation progresses based on a set of simple rules:

  • Life spreads to an empty cell with exactly 3 living neighbors
  • Cells with fewer than 2 living neighbors die from underpopulation
  • Cells with greater than 3 living neighbors die from overpopulation

This simulation is started with randomly placed living cells. Refresh the page to get a different random starting set of cells.
